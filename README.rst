ftw.wfreport
============

``ftw.wfreport`` creates PDF reports of Plone workflows.


Usage
-----

- Add ``ftw.wfreport`` to your buildout configuration:

::

    [instance]
    eggs +=
        ftw.wfreport

- Go to the ``@@workflow-report`` on your Plone site root.
